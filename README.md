<h1 align="center">cb</h1>

## Description

A simple command line utility to copy commonly-used characters to my clipboard if they are difficult to type on a keyboard.

This should also work over SSH, RDP, on WSL, and with X11.

## Quick Start

```shell
$ just

$ ./cb in
∈

$ pbpaste  # on macOS
∈
```
